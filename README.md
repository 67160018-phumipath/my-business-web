# 🌶️ Mala Blaze – เว็บไซต์ธุรกิจหม่าล่าปิ้งย่าง & หม้อไฟเสฉวน

เว็บไซต์นี้เป็นโปรเจกต์สำหรับวิชา Web Development จัดทำเพื่อแสดงตัวอย่างเว็บไซต์ธุรกิจ โดยเลือกธุรกิจ “หม่าล่าปิ้งย่าง & หม้อไฟเสฉวน” พร้อมโครงสร้างหน้าเว็บ 4 หน้า ได้แก่ หน้าแรก, เกี่ยวกับ, บริการ และติดต่อ

เว็บไซต์ใช้โครงสร้าง HTML5, Semantic HTML, และมีการลิงก์หน้าแบบครบถ้วน พร้อมภาพประกอบและฟอร์มติดต่อที่มี validation

## ✔️ ฟีเจอร์ของเว็บไซต์

- Header พร้อมโลโก้และเมนูนำทางทุกหน้า  
- Hero section แบบเน้นข้อความต้อนรับ  
- รายการบริการแบบแยก Section  
- ตารางเปรียบเทียบแพ็กเกจ  
- ฟอร์มติดต่อ + Validation  
- แผนที่ร้านแบบฝัง Google Maps  
- แสดงภาพทีมงานด้วย `<figure>`  
- Footer ครบทุกหน้า  
- รองรับ Accessibility ด้วย alt text
